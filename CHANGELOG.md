# Changelog

DID/VC学習・実装プロジェクトの変更履歴

## [Unreleased] - 2025-09-16

### 新機能追加 (Added)
- **リアカメラQR読み取り機能**
  - html5-qrcode (v2.3.8) ライブラリ導入
  - 認証アプリでのリアカメラ自動起動・QR自動読み取り
  - カメラアクセス権限エラーハンドリング

- **統一ローディングシステム**
  - 共通ローディングライブラリ (`lib/loading.js`) 実装
  - デジタル風SVGスピナーアニメーション
  - VP提示・VC発行申請・再申請時のローディング表示
  - async/await対応による処理制御

### 改善 (Changed)
- **UI/UX統一**
  - 認証アプリとVP提示アプリのボタンデザイン統一
  - プログレッシブディスクロージャーパターン採用
  - ステップ表示削除によるシンプル化

- **コード構造改善**
  - ES6モジュールから通常のscript形式に変更 (CORS問題解決)
  - ローディングCSSをstyle.cssに共通化
  - QRコード生成処理の最適化

### 修正 (Fixed)
- CDN URLの修正（html5-qrcode）
- ローディングオーバーレイ表示問題の解決
- QRコードデータサイズ最適化
- モバイル端末での表示調整

### 技術詳細 (Technical Details)
- **使用ライブラリ**: html5-qrcode v2.3.8, qrcode-generator v1.4.4
- **対応ブラウザ**: モダンブラウザ（カメラAPI対応）
- **実装パターン**: Progressive Disclosure, async/await
- **ファイル構成**: 共通ライブラリ化によるモジュール設計
